# HydLang

The Hydrogen programming language (hydlang) Interpreter.

## Usage:
  
  ### Running Hyd Scripts:
  - **hyd _<script.hyd>_**
  
  ### Interactive mode on command line:
  - **hyd _<no_args>_**
